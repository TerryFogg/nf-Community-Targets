### RP2XXX connection

| GPIO | RP2XXX Function | LCD Connector Pin | Description                         |
| ---- | --------------- | ----------------- | ----------------------------------- |
| GP16 | SPI0_RX         | 25                | D/C - 4 wire interface              |
| GP17 | SPI0_CSn        | 23                | LCD/CS - LCD chip select            |
| GP18 | SPI0_SCK        | 24                | LCD_SCL - 4 Wire                    |
| GP19 | SPI0_TX         | 27                | LCD _SDI                            |
| GP20 | I2C0_SDA        | 31                | I2C_SDA - Capacitive Touch Screen   |
| GP21 | I2C0_SCL        | 30                | I2C_SCL - Capacitive Touch Screen   |
| GP22 | Gpio            | 39                | Interrupt - Capacitive Touch Screen |
| GP26 | Gpio            | 29                | Backlight ON/OFF                    |
| GP27 | Gpio            | 21                | /Reset                              |
| GND  |                 |                   |                                     |
| 3.3V |                 |                   |                                     |
